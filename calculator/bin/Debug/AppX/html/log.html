Logarithm

In order to calculate value of Log(base 10 or e)
1-Input number either by keyboard or
calculator's buttons.
2- Press "Log" for base '10' and "ln" for base 'e'.
Example: Calculate log(10)
input 10 and press log button

In order to use log value of any base number
1- Input base value of log.
2- Then press Logx(y)( 'x' stands for base).
3- Input number to calculate log value
Example: calculate value of log5(10)
Input '5' and then press Logx(y)
input '10' and then press equal('=') button/enter
